완료한 커밋을 수정하기

종종 완료한 커밋을 수정해야 할 때가 있다. 너무 일찍 커밋했거나 어떤 파일을 빼먹었을 때
그리고 커밋 메시지를 잘못 적었을 때 하게 된다. 
다시 커밋하고 싶으면 --amend 옵션을 사용한다:


$ git commit --amend
////////////////////////////////////////// 만약 push가 안된다면 

git push -f origin master
을 써도 되긴하는데 커밋을 잃을 수 있으니 매우 조심해서 사용해야한다(강제 update)
